## 🌟 Các Khối Công Nghệ Cốt Lõi (Core Epics)
- **Kiến trúc Security (RBAC) & Encryption**: Xây dựng khoang Phân quyền Giám Đốc (Admin Dashboard) kiểm soát chặt chẽ lịch sử hệ thống (100% Audit Logs). Dữ liệu mã hoá toàn phần trên bộ nhớ chuẩn AES. Tích hợp Đăng nhập bảo mật đa tầng tối cao (2FA Auth, Lockout, Timeout).
- **Hệ Sinh Thái Phân Tích Tâm Lý (Emotional Tracking)**: Theo dõi và bắt mạch hành vi chi tiêu rủi ro theo tâm lý người dùng thời gian thực (Mua đồ lúc buồn/giận sẽ cảnh báo giảm độ Tín nhiệm Tài chính). Cơ chế Gamification Độc Quyền: Thử thách Không-Tiêu-Tiền (No-Spend Challenge).
- **Cỗ Máy Trí Tuệ (Visual AI OCR)**: Tích hợp thư viện Neural Network cực nặng của `tesseract.js` quét ảnh Chụp Hoá Đơn/Biên Lai, ứng dụng phân tích ký tự tìm kiếm số tiền lớn nhất và điền tự động (Auto-Suggest OCR Data Builder).
- **Giao Thức Tiền Tệ Quần Thể (Split Bill & Social Rank)**: Công thức chia bill tự động và xuất báo cáo đối soát giữa số đông thành viên. Thuật toán So sánh Xếp hạng Tín dụng Chi tiêu trên toàn cầu (VD: Vinh danh Top 15% người dùng hiệu quả nhất hệ thống).
- **Bức Tranh Tổng Cục (Dashboard Analytics)**: Trực quan hoá dòng tiền Thu/Chi 7 ngày gần nhất với sóng LineCharts, cơ chế Import/Export toàn bộ CSDL sang File định dạng dạng khối `.csv` và `.json` để Backup.

## 🚀 Công Nghệ Triển Khai Thực Tế (Tech Stack & Compiler)
- **Lớp Giao Diện Kiến Trúc (Frontend UI)**: Tinh gọn trên nền siêu tảng ReactJS kết hợp hệ module Vite, Vanilla CSS độc quyền (Giao diện chuẩn Ngân hàng số Fintech đa cấu trúc Light/Dark Mode Phủ khối viền tự động đóng mở theo Thời gian).
- **Lõi Đồ Họa Cấu Hình (Chart Engine)**: Thư viện Recharts xử lý và Render dữ liệu DOM Vi mô (LineChart, PieChart, BarChart siêu tương tác).
- **Lõi Thần Kinh Nhận Diện Hình Ảnh**: WebAssembly Tesseract.js.
- **Tiêu Chuẩn Lõi Mã Hoá Tầng Dưới**: Thư viện CryptoJS chuẩn AES bảo vệ thông tin dòng tiền trước mọi đợt tấn công cục bộ.
